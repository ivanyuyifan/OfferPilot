# OfferPilot · AI 面试训练与复盘助手

> 上传简历和目标 JD，AI 基于岗位要求生成针对性模拟面试，每轮回答后给出四维度诊断报告，帮你发现证据缺口、提升表达质量。

🔗 **在线体验**：https://ivanyuyifan.github.io/OfferPilot/

---

## ✨ 核心功能

| 页面 | 做什么 |
|------|--------|
| **岗位分析** | 上传简历 + 粘贴 JD → AI 提取核心职责、生成能力证据地图、推荐 5 道针对性训练问题 |
| **模拟面试** | AI 面试官逐题提问，根据回答质量动态追问（缺少数据/场景时），每题最多追问 2 次后进入诊断 |
| **回答诊断** | 四维度评分（岗位匹配度 / 证据完整度 / 表清度 / 可信度）+ 雷达图 + 有效证据 / 缺失信息 / 改进建议，支持修改回答后重新评分并对比变化 |
| **训练记录** | 能力进步曲线（折线图）+ 历史训练列表 + 80 分以上高分案例收藏 |

## 🎯 产品设计思路

- **不替你生成"完美答案"**：OfferPilot 是训练工具，不是文案生成器。AI 只做诊断和建议，不代写回答
- **证据导向**：四维评分中"证据完整度"占核心位置，引导用户用 STAR 框架组织回答
- **改前改后对比**：修改回答 → 重新评分 → 直观看到分数变化，形成"练习-反馈-迭代"闭环
- **能力追踪**：面试过程中实时标记已考查能力，确保覆盖简历薄弱项

## 🛠 技术栈

- **纯前端单 HTML 文件**，无框架依赖，无后端
- 原生 JavaScript（IIFE 封装，避免全局污染）
- 调用 OpenAI 兼容格式的 LLM API（支持任意兼容服务商）
- **CDN 依赖**：
  - [Chart.js](https://www.chartjs.org/) — 雷达图 & 折线图
  - [html2canvas](https://html2canvas.hertzen.com/) + [jsPDF](https://github.com/parallax/jsPDF) — PDF 报告导出
- **数据存储**：localStorage（训练记录 + API 配置，不上传任何数据到服务器）
- **设计**：CSS 变量驱动暗色模式，响应式适配桌面 / 平板 / 手机

## 🚀 快速开始

1. 打开 [在线 Demo](https://ivanyuyifan.github.io/OfferPilot/)
2. 点击右上角 **⚙** 配置 API：
   - **Endpoint**：填入兼容 OpenAI 格式的聊天补全地址（如 `https://api.openai.com/v1/chat/completions`）
   - **API Key**：填入服务商提供的密钥
   - **模型**：填入模型名称（如 `gpt-4o`、`glm-4-plus` 等）
3. 粘贴简历和目标岗位 JD → 点击「开始分析」
4. 查看能力地图后进入模拟面试 → 逐题回答
5. 完成后查看诊断报告，可修改回答重新评分

> 💡 也可以点击右上角 **?** 查看完整使用手册

## 📂 本地运行

```bash
git clone https://github.com/ivanyuyifan/OfferPilot.git
cd OfferPilot
# 直接用浏览器打开 index.html，或启动本地服务器：

```

## 📋 诊断维度说明

| 维度 | 评估什么 |
|------|----------|
| 岗位匹配度 | 回答是否回应了岗位 JD 的核心要求 |
| 证据完整度 | 是否包含具体场景、行动和量化结果 |
| 表达清晰度 | 结构是否清楚、重点是否突出 |
| 可信度 | 细节是否具体、一致且经得起追问 |

## 🔒 隐私说明

- 所有数据（简历、JD、回答、训练记录）仅存储在当前浏览器的 localStorage 中
- 不上传到任何服务器，不收集任何用户信息
- 清除浏览器数据会同时清除所有训练记录，请注意备份
